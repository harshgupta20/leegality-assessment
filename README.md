# 🛍️ Leegality Store 
 
A modern ecommerce frontend built with React, React Router, Axios and Tailwind CSS.
 
🌐 Live Demo: https://leegality-assessment.web.app/ 
 
---
 
## ✨ Features 
 
- Product Listing
- Product Details Page
- Search Products
- Category Filtering
- Sorting
- Pagination
- URL State Persistence
- Responsive Design
- Loading States
